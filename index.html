30 DAYS OF CODE

A coding challenge inspired by Jennifer Dewalt where I am going to code every day for a month starting July 21, 2017.

The aim of this challenge is to learn a new design trick everyday for the next thirty days. If you want to do this challenge, feel free to do so! This is not for complete beginners in web design but for people with basic knowledge of HTML, CSS and JavaScript.
